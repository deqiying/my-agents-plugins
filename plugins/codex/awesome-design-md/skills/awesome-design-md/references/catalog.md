# Awesome DESIGN.md Local Catalogue

Use this file as the fast local index for the vendored `awesome-design-md` collection.

Important:

- The upstream repository is a catalogue and many per-brand `README.md` files are redirect stubs.
- Use this catalogue to choose a slug.
- Fetch the real `DESIGN.md` into the target project with `..\scripts\fetch-design.ps1` or `npx -y getdesign@latest add <slug>`.
- Use `references\upstream-repo\README.md` when you need the original upstream wording.
